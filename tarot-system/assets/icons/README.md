# 아이콘 폴더

타로카드 시스템에서 사용하는 아이콘들을 저장하는 폴더입니다.

## 포함 아이콘
- `logo.png`: RHEIGHT 로고
- `tarot-icon.png`: 타로카드 대표 아이콘
- `spread-icons/`: 각 스프레드별 아이콘들
  - `daily.png`: 일일 운세 아이콘
  - `love.png`: 사랑 운세 아이콘
  - `career.png`: 진로 운세 아이콘
  - `celtic.png`: 켈트 십자 아이콘

## 아이콘 사양
- 권장 크기: 64x64px, 128x128px
- 파일 형식: PNG (투명 배경)
- 색상: 금색(#ffd700) 계열
