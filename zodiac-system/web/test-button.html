🏠 홈 버튼 테스트 페이지

현재 URL:
현재 경로:

1. 수정 전 버튼 (원본)

<button onclick="location.href='../../index.html'">홈으로 (수정 전)</button>

2. 수정 후 버튼 (현재)

<button onclick="location.href='/index.html'">홈으로 (수정 후)</button>

3. 다양한 경로 테스트

<button onclick="location.href='/'">루트로</button>
<button onclick="location.href='../index.html'">상위 디렉토리 index</button>
<button onclick="window.location.href='/'">window.location 사용</button>
<button onclick="window.location.pathname='/'">pathname 사용</button>

4. JavaScript 함수 사용

function goToHome() {
  window.location.href = '/';
}
function goToIndex() {
  window.location.href = '/index.html';
}

5. 절대 경로 테스트

<button onclick="location.href='http://localhost:8080/'">절대 경로</button>
<button onclick="location.href='http://localhost:8080/index.html'">절대 경로 index</button>
